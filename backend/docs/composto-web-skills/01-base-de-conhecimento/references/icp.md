# ICP Composto Web: Detalhamento Completo

## ICP PRIMÁRIO

| Dimensão | Perfil |
|---|---|
| Porte | 100 a 1.000 funcionários |
| Faturamento | R$10M a R$500M/ano |
| Localização | Todo o Brasil (operação remota) |
| Setores prioritários | Financeiro, Jurídico, Saúde, Indústria, Varejo/E-commerce, TI |
| Maturidade tecnológica | Usa ERP/CRM mas com baixa integração entre sistemas |
| Momento ideal | Crescimento acelerado OU ineficiência que já comprometeu resultado |
| TI interna | 1-5 pessoas sem capacidade para projetos complexos |
| Decisor | CEO, CTO, CFO, COO com autonomia de R$20k+ |
| Ticket mínimo | Setup R$20.000 + mensalidade R$3.000/mês |

**Sinais de que é o cliente certo:**
- Fala "não consigo escalar sem contratar mais gente"
- Sistemas desconectados gerando retrabalho visível e mensurável
- Já tentou resolver internamente e não conseguiu
- Urgência real: o problema já está custando dinheiro ou gerando risco
- Valoriza ser dono do que vai ser construído

**Sinais de alerta (pode ser o cliente errado):**
- Busca o mais barato, compara com freelancer sem critério técnico
- Não tem clareza do problema, só da solução que imagina
- Espera que a Composto Web assuma a gestão do processo de negócio
- Decisão depende de 5+ aprovações sem prazo claro

## ICP SECUNDÁRIO (base atual PME)

| Dimensão | Perfil |
|---|---|
| Porte | 10 a 100 funcionários |
| Faturamento | R$1M a R$10M/ano |
| Setores | Advocacia, clínicas, pequeno varejo, prestadores de serviço |
| Decisor | Dono ou sócio operando no operacional |
| Ticket | Setup R$3.000-R$19.000 + mensalidade R$800-R$2.999 |
| Produto ideal | Agentes IA no WhatsApp, automações pontuais, integrações simples |

**Estratégia:** atender bem com produto padronizado. Usar como pipeline de upsell quando crescer.

---

## ANTI-ICP

| Perfil | Motivo |
|---|---|
| Quer só site ou app sem complexidade | Fora do escopo de valor |
| Startup early-stage sem produto validado | Ticket incompatível, escopo muda todo sprint |
| Quer entrega pontual sem parceria | Modelo de evolução contínua é parte do valor |
| Quer substituir 100% do contato humano por IA sem curadoria | Expectativa irreal, gera desgaste |
| Budget abaixo de R$5.000 | Custo de venda e entrega inviabiliza margem |

## GATILHOS DE PROSPECÇÃO

Momentos em que o ICP primário está mais receptivo:

1. **Crescimento acelerado:** contrataram muito nos últimos 12 meses, operação caótica
2. **Troca de ERP ou CRM:** janela perfeita para construir a camada de integração certa
3. **Fusão ou aquisição:** dois sistemas precisam se conversar com urgência
4. **Novo produto ou canal:** e-commerce, nova unidade, novo mercado
5. **Regulação ou auditoria:** LGPD, ISO, auditoria de cliente grande
6. **Saída de pessoa-chave de TI:** o "cara que sabia tudo" foi embora
7. **Problema público:** sistema caiu ou dados incorretos geraram problema com cliente estratégico
